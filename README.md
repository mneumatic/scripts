# Personal Scripts

My personal collection of shell scripts, `.destkop` files and associated icons & files.

## Podman Containers

This will run a series of shell scripts to install podman containers I use on the regular along with desktop launchers.

```
cd podman
chmod +x run.sh # If needed
./run.sh
```

### Requires

- Linux - However someone might be able to use some of these scripts in WSL. Not tested on Windows or MacOS.
- Podman 5.8.1+
- Web Browser (Not you GNOME Web)
